**Temperature Conversion Tool**

A CLI temperature converter written in C++ that converts temperatures between Farenheit, Celsius and Kelvin

## Features
- Converts temperatures between Farenheit, Celsius and Kelvin
- Menu-driven interface with 7 different conversion options
- Uses input validation and handles erros
- Uses Do-while loop to loop continously until user explicitly exits program

 ** How to Run **

g++ main.cpp -o temp_converter && ./temp_converter
